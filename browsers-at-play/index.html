Creating CSS Animation


@keyframes flashing {
  0%   { opacity: 0; }
  50%   { opacity: 1; }
  100% { opacity: 0; }
}

.lights i {
	animation:flashing 2s infinite;
}

You'll need vendor prefixes :)

List of animatable properties: http://oli.jp/2010/css-animatable-properties/

You can do lots of things with the web audio API. Simple start/stop volume More complex things like add filters to the wave form You can even create your own sounds THE O'REILLY BOOK IS FREE! For my POC I want to focus on analysing the waveform ->

Analysing

We can do that :D


//create instance
var analyser = myAudioContext.createAnalyser();

//get data
analyser.getByteFrequencyData();
analyser.getFloatFrequencyData();
analyser.getByteTimeDomainData();
You can analyse an audio context using the web audio api by creating an analyser node That then gives you methods for frequency and time data - brilliant just what we need I want frequency
